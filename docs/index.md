# SIGEsporte

**Sistema de Gestão de Eventos Esportivos**

O SIGEsporte é uma plataforma web desenvolvida para facilitar a organização, divulgação e inscrição em eventos esportivos universitários. O sistema conecta alunos, organizadores e gestores de espaços esportivos em um ambiente centralizado e de fácil uso.

---

## Sobre o Projeto

O projeto é desenvolvido como trabalho da disciplina de **Métodos de Desenvolvimento de Software (MDS)**, ministrada pelo professor Ricardo Ajax na Universidade de Brasília (UnB), pelo grupo **Dijkstra**.

**Repositório:** [github.com/FGA0138-MDS-Ajax/2026.1-T01-Dijkstra](https://github.com/FGA0138-MDS-Ajax/2026.1-T01-Dijkstra)

---

## Perfis de Usuário

O sistema atende três tipos de usuários:

- **Aluno** — visualiza e se inscreve em eventos esportivos
- **Organizador** — cria e gerencia eventos, controla inscrições e reservas de espaços
- **Gestor** — administra os espaços esportivos e aprova reservas

## Navegação

Use o menu lateral ou superior para acessar:

- **Atas de Reunião** — registros das reuniões de equipe
- **Documento de Visão** — escopo e objetivos do produto
- **Documento de Arquitetura** — decisões técnicas e estrutura do sistema
- **Diagramas** — diagramas de classes, arquitetura, sequência e dados
- **Protótipos** — protótipos de baixa fidelidade das telas
