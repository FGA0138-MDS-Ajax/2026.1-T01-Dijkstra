# Guia de Implantação de Infraestrutura

## Objetivo

Este guia apresenta o processo completo para provisionar, configurar e publicar serviços em uma VPS (Virtual Private Server).

Ao final do roteiro, a infraestrutura estará preparada para hospedar aplicações web, bancos de dados e serviços auxiliares acessíveis pela Internet.

O processo foi dividido em etapas independentes para facilitar a reutilização da documentação em diferentes cenários de implantação.

## Escopo

Este guia aborda:

- Escolha e criação de uma VPS;
- Configuração inicial do sistema operacional;
- Controle de acesso e segurança básica;
- Instalação de ferramentas de administração;
- Instalação de dependências de infraestrutura;
- Configuração de serviços auxiliares;
- Publicação de aplicações na Internet;
- Configuração de domínios e certificados digitais.
