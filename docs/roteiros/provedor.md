# Roteiro Inicial — Escolha e Criação de VPS Gratuita

## Objetivo

Este guia tem como objetivo auxiliar na escolha de uma plataforma de hospedagem para implantação de aplicações, serviços e projetos acadêmicos utilizando recursos gratuitos permanentes (*Always Free*) ou benefícios disponíveis para estudantes.

---

## Etapa 1 — Escolha o Serviço de VPS

Antes de iniciar a criação do servidor, escolha a plataforma mais adequada ao seu cenário.

### Cenários Recomendados

| Cenário | Provedor Recomendado |
|----------|----------|
| Maior quantidade de recursos gratuitos permanentes | Oracle Cloud |
| Ambiente corporativo amplamente utilizado no mercado | AWS |
| Integração com ecossistema Microsoft | Azure |
| Aprendizado de infraestrutura Google | Google Cloud |
| Projetos acadêmicos com benefícios estudantis | GitHub Student Pack + Azure/AWS |
| Facilidade de uso e documentação | DigitalOcean (via benefícios estudantis) |

---

# Resumo dos Prós e Contras

## Oracle Cloud

### Prós
- Melhor plano gratuito permanente disponível atualmente.
- Grande quantidade de RAM e CPU.
- Excelente para projetos pessoais e acadêmicos.
- Permite execução de múltiplos serviços simultaneamente.

### Contras
- Processo de cadastro pode ser rigoroso.
- Interface menos intuitiva.
- Menor presença em ambientes corporativos.

---

## AWS

### Prós
- Maior participação de mercado.
- Grande quantidade de serviços.
- Excelente para aprendizado profissional.

### Contras
- Curva de aprendizado elevada.
- Benefícios gratuitos geralmente temporários.
- Plataforma complexa para iniciantes.

---

## Azure

### Prós
- Forte integração com tecnologias Microsoft.
- Excelente programa para estudantes.
- Muito utilizado em empresas.

### Contras
- Interface extensa.
- Recursos gratuitos limitados.

---

## Google Cloud

### Prós
- Excelente infraestrutura global.
- Boa integração com ferramentas Google.
- Forte presença em dados e inteligência artificial.

### Contras
- Camada gratuita de VPS bastante limitada.
- Menos opções gratuitas permanentes.

---

## DigitalOcean

### Prós
- Plataforma simples.
- Documentação de alta qualidade.
- Ótima para iniciantes.

### Contras
- Não possui VPS gratuita permanente.
- Dependência de créditos promocionais ou estudantis.
